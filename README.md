This Ai based stock price prediction model project has been developed using different python libraries and frameworks (Written in requirements.txt file).

To run this code:
  -> install all the libraries in the requirements file inside virtual env. in pycharm or your editor
  -> After successful installation run the file in the terminal by command <streamlit run (filename).py>
  -> for better result increase the epoch numbers
  -> Enter the stock name in the bar and it will display the result
